# MeetInsights – Анализ видеозвонков
Платформа для автоматического анализа корпоративных видеозвонков с помощью технологий speech-to-text\ и обработки естественного языка (NLU). Сервис позволяет быстро получать текстовую расшифровку,\ оценивать тональность обсуждений и выявлять ключевые темы.

## Возможности
1.Загрузка видеофайлов (поддерживается формат MP3)\
2.Автоматическая транскрибация речи в текст с использованием модели Whisper\
3.Определение тональности разговора: позитивная, негативная или нейтральная\
4.Извлечение ключевых слов из текста (до 3 наиболее значимых терминов)\
5.Возврат структурированного результата в виде текстового отчёта\
## Технологии
1.**FastAPI** – фреймворк для создания веб-API\
2.**Whisper** – модель транскрибации от OpenAI\
3.**NLTK (VADER)** – анализ тональности текста\
4.**rutermextract** – извлечение ключевых слов на русском языке\
5.**HTML/JavaScript** – простой клиентский интерфейс для загрузки файлов
